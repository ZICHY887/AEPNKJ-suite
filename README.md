**Impulse After Effects Extension & After Effects Graph Editor**

Welcome to the official repository for the AEPNKJ suite of professional-grade extensions and native plugins for Adobe After Effects.

## Overview
AEPNKJ tools are designed to push Adobe After Effects beyond its absolute limits, accelerating your creative workflow.

### ⚡ Impulse After Effects Extension
The ultimate productivity powerhouse. Packed with game-changing modules like Smart Interchange, Advanced Speed Influence, and instant cache nuking. 
- **Smart Interchange:** Mathematically calculate and scale replaced layers.
- **Cache Nuke:** Instantly clear RAM and Disk Cache.
- **Workflow Shortcuts:** Advanced modifier keys for layer pre-composing and trimming.

### 🎨 Catalyst Native Plugin
A truly bulletproof native plugin built on C++. Features a universal plugin scanner, custom hotkeys, and a built-in snapshot gallery.
- **Universal Scanner:** Instantly indexes your entire effects library.
- **God Tier UI:** Hardware accelerated interface (DirectX 11 & ImGui).
- **Auto-Save Data:** Seamlessly saves your preferences and hotkeys.

### 🌪 Wind After Effects Graph Editor
An advanced interactive flow graph editor. Features endless presets, custom save states, and an ultra-fluid UI to revolutionize your animation curves.
- **Total Control:** A fluid interface designed specifically for animation precision.
- **Preset Master Library:** Pre-built graph presets for instant, perfect ease curves.

## How to Install
1. **Download** the relevant installer from our website (or execute the `.exe` files if downloaded locally).
2. **Extensions (Impulse & Wind):** Place the files into your Adobe CEP extensions folder (`C:\Program Files (x86)\Common Files\Adobe\CEP\extensions\` on Windows).
3. **Registry Fix:** Ensure `PlayerDebugMode` is set to `1` in your Windows Registry under `HKEY_CURRENT_USER/Software/Adobe/CSXS.11`.
4. **Native Plugins (Catalyst):** Run the automated installer which places files in your Adobe MediaCore directory.
