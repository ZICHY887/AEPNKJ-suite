IMPULSE DOCUMENTATION

Operational Manual for AEPNKJ IMPULSE V3.0. Read before deployment.

01. INSTALLATION PROTOCOL

Impulse V3 is an unsigned extension. You must install it manually into the Adobe CEP directory.

Windows Path:

C:\Program Files (x86)\Common Files\Adobe\CEP\extensions\

Mac OS Path:

/Library/Application Support/Adobe/CEP/extensions/

1. Extract the downloaded folder.
2. Move the entire com.aepnkj.impulse folder into the path above.
3. Restart Adobe After Effects.

02. CRITICAL: DEBUG MODE FIX

⚠️ SYSTEM ALERT: If the extension panel is blank or buttons don't work, you must enable "PlayerDebugMode".

Since this is a custom tool, Windows requires a registry key to allow it to run.

One-Click Fix (Windows):

1. Press Win + R
2. Type regedit
3. Navigate to: HKEY_CURRENT_USER/Software/Adobe/CSXS.11 (or your version)
4. Right Click > New > String Value
5. Name: PlayerDebugMode | Value: 1

03. INTERFACE OVERVIEW

The panel is divided into three strategic sectors:

  • MAIN: Core animation tools (Ease, Anchor, Wiggle).
  • TOOLS: Layer manipulation (Cuts, Duration, Swapper).
  • SYS: Settings, Visual Themes, and Cache Nuke.

04. CORE MODULES

⚡ Speed Influence

Replaces the Graph Editor. Enter your desired Influence % (e.g., 75) and click Apply. The system will auto-detect if your keyframes are Linear and warn you.

🔄 Smart Interchange

The crown jewel of V3. Select a layer, click Swap, and choose a new file. Impulse will replace the source but mathematically calculate the scale to match the previous layer's visual size exactly.

💾 Cache Nuke

Clears RAM and Disk Cache instantly without opening preferences. Use this when After Effects starts lagging.

PRO TIP: Use the "Auto-Fit" module to instantly resize massive images to fit your composition size.

05. WORKFLOW SHORTCUTS

Impulse supports modifier keys for advanced functionality:

  • Smart Pre-Comp (Click): Pre-composes each selected layer individually.
  • Smart Pre-Comp (Shift + Click): Merges all selected layers into ONE pre-comp.
  • Trim In/Out: These buttons respect the Playhead (CTI) location accurately.

CATALYST DOCUMENTATION

Operational Manual for Catalyst Plugin. Read before deployment.

06. INSTALLATION PROTOCOL

Unlike Impulse, Catalyst is a native .aex plugin and comes with an automated setup wizard.

PRO TIP: Simply run the catalyst_Installer.exe file. It will automatically locate your Adobe MediaCore folder and install the plugin globally for all versions of After Effects.

Manual Verification Path (Windows):

C:\Program Files\Adobe\Common\Plug-ins\7.0\MediaCore\catalyst\

07. CORE CAPABILITIES & MODULES

🚀 Core Engine & Stability

Bulletproof Architecture: Built entirely on C++ and a JS Bridge using an "Async Queue System" (IdleHook). This guarantees that opening the UI or applying effects will never cause After Effects to freeze, deadlock, or crash. It is 100% stable and indestructible.

Universal Scanner: Instantly indexes your entire library. It automatically pulls all Native Effects, 3rd Party Plugins (Sapphire, BCC, Deep Glow, etc.), and custom .ffx Presets from your Documents/Program Files into one unified search engine.

🎨 Premium "God Tier" UI

Hardware Accelerated: Powered by DirectX 11 and ImGui, ensuring the interface launches instantly with absolutely zero lag.

Cyberpunk Aesthetics: Features a sleek translucent dark glassmorphism background (with a Light Mode option), highlighted by a custom glowing Cyan-to-Purple gradient separator for a premium Mac-Spotlight feel.

Dynamic Customization: Use live sliders to adjust the Window Width and UI Scale (text/button sizes) to perfectly fit your workspace.

⚡ Workflow Enhancements

Instant Apply: Search your desired effect and simply hit Enter or click. The effect is applied to your selected layer instantly.

Custom Hotkey Trigger: Don't be limited by default shortcuts. Bind Catalyst to open with Tab, Space, Enter, or set a completely custom key combination (like F3 or Shift+A) from the dropdown.

⭐ Smart Favorites Manager: Pin your most frequently used effects so they always stay at the top. Easily manage them using the Auto-Suggest bar located in the Settings tab.

📷 Snapshot & Gallery System: Click [ SNAP ] to communicate directly with the AE render engine and save a high-quality .png of your current frame without any workflow interruption. Click [ GLRY ] to instantly open your Windows save directory.

💾 Auto-Save Data System: Your preferences are never lost. UI size, themes, hotkeys, and favorites are automatically saved to a Catalyst_Settings.ini file in your Documents folder. Every time you restart AE, everything is exactly how you left it.
