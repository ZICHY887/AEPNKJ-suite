AEPNKJ Suite Manual

System Architecture Overview

The AEPNKJ Suite is a professional-grade ecosystem designed to accelerate motion workflows inside Adobe After Effects. It consists of two structural formats:

CEP Extensions

Impulse v7 & Wind v3 run on Adobe's CEP (Common Extensibility Platform) framework. They integrate directly into AE's panel system and use HTML5/JS frontend bridges connected to ExtendScript (JSX) rendering cores.

Native Plug-ins

Catalyst v3 is a native C++ compiled binary (.aex). It hooks directly into the After Effects engine, providing high-performance GPU-accelerated layouts (DirectX 11 & ImGui) and instant system access without extension bottlenecks.

AE Extension Suite

Impulse v7 (Workflow Extension)

Impulse is the ultimate productivity and automation extension. Everything is built-in directly to make your everyday After Effects workflow infinitely faster. Rather than navigating deep preference menus, Impulse wraps everything you need in an offline-capable, lightweight workspace.

Core Modules Included:

  • Speed Influence: Instantly apply precise easing percentages to multiple selected keyframes.
  • Smart Interchange: Scale and fit newly swapped assets automatically based on original layout dimensions.
  • Cache Nuke: Clear disk and RAM caches in a single click, keeping your system fast.
  • Auto-Fit Canvas: Rescale media objects to fit composition boundaries instantly.

Impulse Installation Path

Since Impulse is a CEP Extension, you must place the extracted folder manually inside Adobe's extensions directory.

Windows Destination:
C:\Program Files (x86)\Common Files\Adobe\CEP\extensions\com.aepnkj.impulse
macOS Destination:
/Library/Application Support/Adobe/CEP/extensions/com.aepnkj.impulse
PlayerDebugMode Requirement

Unsigned CEP extensions will fail to load or show blank panels in After Effects by default. You must configure Adobe CSXS debug mode to enable them.

Enabling Debug Mode on Windows:

  1. Press Win + R, type regedit, and hit Enter.
  2. Navigate to HKEY_CURRENT_USER\Software\Adobe\CSXS.11 (or whichever CSXS versions correspond to your After Effects install, like CSXS.9 to CSXS.11).
  3. Right-click in the right pane, select New > String Value.
  4. Name the value PlayerDebugMode.
  5. Double-click it and set its Value Data to 1.
Native C++ Plugin

Catalyst v3 (Effect Search Plugin)

Catalyst is a lightning-fast native plugin that serves as a highly functional, search-on-the-fly console engine for all your AE effects and presets. Designed to keep editors in the zone, Catalyst operates with a background async queue framework that guarantees After Effects will never experience UI freezing or locking when scanning plugins.

Key Features:

  • Instant Universal Scanner: Indices internal effects, third-party plug-ins (Sapphire, BCC, Red Giant), and user preset paths simultaneously.
  • ImGui DirectX 11 Interface: Ultra-responsive, GPU-accelerated floating search bar.
  • Favorites System: Fast key bindings and quick-pin filters to access regular assets immediately.
  • Snapshot Capture: Built-in canvas export to capture PNG thumbnails directly to folders.

Catalyst Installation Path (Native Plugin)

Catalyst is a native binary plugin (not an extension). Run the catalyst_Installer.exe setup manager to place the C++ plugin files directly into Adobe's shared MediaCore directory automatically. If performing manual deployment, copy the .aex plugin folder here:

C:\Program Files\Adobe\Common\Plug-ins\7.0\MediaCore\catalyst\
AE Graph Extension

Wind v3 (Ease Curve Graph Extension)

Wind is a professional graph easing editor extension designed specifically to bypass the limitations of the default After Effects graph workspace. Built around a five-engine curve architecture (Bezier, Elastic, Steps, Bounce, Wave), Wind lets you generate precise keyframe velocities dynamically with smooth easing curves and real-time previews.

Wind Installation Path

Like Impulse, Wind v3 is a CEP Extension and resides in the standard extensions directories:

Windows Destination:
C:\Program Files (x86)\Common Files\Adobe\CEP\extensions\com.aepnkj.wind
